## Program Overview

The Multiplatform Application Development program (Desarrollo de Aplicaciones Multiplataforma - DAM) is a vocational training qualification that provides comprehensive education in modern software development practices.

    **Database Systems**
    - Relational database design
    - SQL and database management
    - Data modeling and normalization
    - Database administration basics
    
    **Markup Languages and Information Systems**
